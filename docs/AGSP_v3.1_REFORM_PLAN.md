**作成日:** 2026年1月31日  
**作戦コード:** AGSP-REFORM-v3.1  
**状態:** リポジトリ更新完了

---

## 📋 改革概要

### 基本方針（幕僚長承認済み）

1. **参謀本部を3つのAIで構成**
   - Claude Desktop: 参謀総長（情報統括）
   - Gemini: 作戦部長（作戦計画立案）
   - ChatGPT: 情報部長（情報分析・リスク評価）

2. **現場をClaude Code（ECC）方面軍で運用**
   - 方面軍方式で機能ごとに完結
   - git worktree による並列実行
   - FrontEnd方面、BackEnd方面など

3. **情報共有方式**
   - Claude DesktopがMCP経由でGitHubリポジトリを参照
   - ChatGPT/Geminiは断片情報のみ（README.md, CURRENT_STATUS.md）
   - 深い情報はClaude Desktopが整理・要約

4. **指示伝達方式**
   - ファイルに情報を記載
   - 短いプロンプトで「○○を読め」と指示
   - 人間がコピペで仲介（情報欠落を防ぐ）

## 📋 役職対応表

| 役職名 | 担当 | 職務 | 史実の役割 |
|-------|------|------|-----------|
| **幕僚長** | User | 最終裁可、方針決定 | 天皇（大元帥）に相当 |
| **参謀総長** | Claude Desktop | 全情報集約、戦略統合、最終作戦立案 | 参謀本部の最高責任者 |
| **作戦部長** | Gemini | 作戦計画立案、実装方針、技術選定 | 第一部（作戦部）の責任者 |
| **情報部長** | ChatGPT | 情報収集・分析、リスク評価、代替案 | 第二部（情報部）の責任者 |
| **連合艦隊司令長官** | Claude Code（統括） | 全方面軍の統括、作戦実行指揮 | 現場最高司令官 |
| **方面軍司令官** | Claude Code（各worktree） | 担当領域の実装実行 | 地域・機能別の独立部隊長 |
